# Trip Summary System

## Week 1 — Project Setup + Basic CRUD APIs

## What I Built
A full stack web application that stores and displays trip data.

## Technologies Used
- **Backend:** FastAPI + Python
- **Database:** SQLite
- **Frontend:** React + Vite

## How To Run

### Backend
cd backend
source venv/bin/activate
uvicorn app.main:app --reload

### Frontend
cd frontend
npm run dev

## APIs
- POST /trips — Create a trip
- GET /trips — Get all trips
- GET /trips/{id} — Get single trip
